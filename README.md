# Prompt Optimizer

AI-powered Prompt Iteration Optimization Tool

## Installation

```bash
pip install -e .
```

## Usage

### CLI
```bash
python -m prompt_optimizer.cli --prompt "your prompt" --test-data test.json
```

### Web Interface
```bash
python web_app.py
```

## Development

```bash
pip install -e ".[dev]"
pytest
```
